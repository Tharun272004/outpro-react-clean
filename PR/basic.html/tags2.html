  1. it is a keyboard input
  2. Used to represent keys you press on the keyboard.

fieldset tag

  1. The fieldset tag is used to group related elements in a form.
  2. The fieldset tag draws a box around the related elements.

Datalist

  1. The datalist tag specifies a list of pre-defined options for an input element.
  2. he datalist tag is used to provide an "autocomplete" feature for input elements. Users will see a drop-down list of pre-defined options as they input data.
